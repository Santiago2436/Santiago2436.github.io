# Calculadora de Aportes – Empleados Dependientes

Proyecto desarrollado en HTML, CSS y JavaScript que permite calcular los aportes
a seguridad social de un empleado dependiente en Colombia.

## Tecnologías
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Cálculo de salud, pensión, ARL y caja de compensación
- Validación del salario mínimo legal vigente
- Diseño responsive
